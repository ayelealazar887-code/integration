# Full Stack React + Node.js + PostgreSQL App

This is a simple full-stack application built using:

- React (Frontend)
- Node.js + Express (Backend)
- PostgreSQL (Database)

It fetches user data from a PostgreSQL database and displays it in a React table.

---

## 🚀 Features

- Fetch users from PostgreSQL database
- Display data in React table
- REST API using Express
- Clean separation of frontend and backend

---

## 🛠️ Tech Stack

**Frontend:**
- React
- JavaScript
- Vite

**Backend:**
- Node.js
- Express
- PostgreSQL
- pg (node-postgres)
